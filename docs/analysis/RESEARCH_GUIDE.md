## Gene-Tree Core Features (For Comparison)

Based on MASTER_PLAN.md:

### Philosophy
1. Privacy-first - Every data field has privacy controls
2. Cultural awareness - Respect kinship complexity
3. Verification-based trust - Two-way relationship confirmation
4. Preservation over perfection - Handle uncertain data
5. Stories matter - Not just names and dates

### Current Features
- Authentication (Supabase)
- User profiles with privacy settings
- Relationships (parent/child/spouse/sibling + qualifiers)
- Kinship computation
- Tree visualization (D3.js + ELK.js)
- Invitation system (token-based email)
- Stories (text/photo) with moderation
- Internationalization (EN/RU)
- Reactions system
- Threaded comments with @mentions
- Activity feed
- "This Day" widget
- Photo tagging
- Memorial tributes
- Relationship finder
- Ask the Elder Q&A

### Target Pricing (Gene-Tree planned)
- Free tier: 50 profiles
- Premium: $7.99/mo
- Family: $14.99/mo
